## ğŸ” CIA Triad â€“ My Understanding

### Confidentiality  
I learned that confidentiality ensures sensitive data is accessed only by authorized users.  
Examples include encrypted banking transactions and private messaging applications.

### Integrity  
Integrity ensures that data remains accurate and unchanged unless modified by authorized users.  
This is critical in financial transactions and system logs.

### Availability  
Availability ensures that systems and services remain accessible whenever needed.  
Attacks like DDoS and ransomware mainly target availability.
